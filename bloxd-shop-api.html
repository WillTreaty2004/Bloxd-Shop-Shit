Shop API

Control the in-game shop (opened with B) to create purchasable items, collectibles, menus, and player-specific experiences.

📌 The shop is opened in-game by pressing B. You can control categories, items, pricing, user input, and per-player overrides entirely through scripts.

Quick Start

// Create a simple item in a category
api.createShopItem("myGame:shop", "ironSword", {
  image: "Iron Sword",
  customTitle: "Iron Sword",
  description: "A trusty iron sword.",
  cost: 100,
  currency: "Gold Coin",
  canBuy: true,
  onBoughtMessage: "You got an Iron Sword!"
});

// Listen for purchases
api.onPlayerBoughtShopItem = (playerId, categoryKey, itemKey, item, userInput) => {
  if (itemKey === "ironSword") {
    giveItem(playerId, "Iron Sword", 1);
  }
};

Key Points

  • Categories are auto-created when you call createShopItem
  • Per-player overrides let you show different items/prices to different players
  • createShopItem replaces any existing item with the same key
  • userInput lets players type text, pick from a dropdown, or select a player
  • Items can be hidden, flagged with red dots, or given "new" / "lucky" badges
🧠

Core Concepts

Understand keys, categories, and the override system.

CategoryKey & ItemKey

Both are plain strings. Convention is to namespace them like "gameMode:section" to avoid collisions with other systems.

// Examples of keys
"classicGame:utilities"   // built-in bloxd category
"myGame:weapons"          // custom category
"myGame:cosmetics"

Override Priority

When a player opens the shop, item properties are resolved in this order (highest wins):

Per-player override
highest
Global item
default
Engine defaults
lowest

Item Lifecycle

// 1. Create global item (all players see it)
api.createShopItem(categoryKey, itemKey, item);

// 2. Override for a specific player
api.updateShopItemForPlayer(playerId, categoryKey, itemKey, { cost: 50 });

// 3. Reset the player back to the global version
api.resetShopItemForPlayer(playerId, categoryKey, itemKey);

// 4. Remove the global item (also removes all player overrides)
api.deleteShopItem(categoryKey, itemKey);

userInput Types

Collect data from the player before they confirm a purchase.

Text Input

userInput: {
  type: "text",
  placeholderText: "Enter a name...",  // optional
  wordCharsOnly: true               // optional, restrict to word chars
}

Dropdown

userInput: {
  type: "dropdown",
  dropdownOptions: ["Option A", "Option B", "Option C"]
}

Player Selector

userInput: {
  type: "player",
  excludedPlayers: ["-3"]   // optional: playerIds to hide from the list
}

Accessing Input in Callback

api.onPlayerBoughtShopItem = (playerId, categoryKey, itemKey, item, userInput) => {
  if (itemKey === "renameThing") {
    rename(playerId, userInput); // userInput = whatever the player typed/selected
  }
};
🎨

CustomTextStyling

Rich, mixed-content text for titles, descriptions, and button labels.

Instead of a plain string, text fields accept an array of segments — each can be a string, a translation key, or an icon.

Segment Types

// Translation key only
customTitle: [{ translationKey: "tribe:createTribe" }]

// Mix of translated text + icon + plain string
buyButtonText: [
  { str: { translationKey: "general:create" } },
  { str: ": " },
  { icon: "Iron Bar" },
  { str: " 2" }
]

// Plain string still works everywhere
description: "A simple description."
💡 Mix translated keys and plain strings in the same array for fully localized rich text UI.

Global Methods

Manage shop items and categories visible to all players.

api.createShopItem(categoryKey, itemKey, item)

Create or replace a global shop item. Creates the category if it doesn't exist. Deletes any per-player overrides for the same keys.

ShopCategoryKeyShopItemKeyShopItem
api.updateShopItem(categoryKey, itemKey, changes)

Patch specific properties of an existing global item. Throws if item doesn't exist.

Partial<ShopItem>
api.deleteShopItem(categoryKey, itemKey)

Delete a global item and ALL per-player overrides for it. Throws if item doesn't exist.

⚠ destructive
api.configureShopCategory(categoryKey, config)

Set display properties on a category (title, sort order, red dot, auto-select, description).

ShopCategoryConfig
// Hide an item then reveal it later
api.createShopItem("myGame:shop", "secretItem", {
  image: "Diamond",
  hidden: true,
  canBuy: false
});

// Later, reveal it with a red dot
api.updateShopItem("myGame:shop", "secretItem", {
  hidden: false,
  canBuy: true,
  redDot: true
});
👤

Per-Player Methods

Override shop items and categories for individual players.

api.createShopItemForPlayer(playerId, categoryKey, itemKey, item)

Create a fully custom item visible only to this player. Replaces any existing override.

api.updateShopItemForPlayer(playerId, categoryKey, itemKey, changes)

Update specific properties for one player only. Throws if item doesn't exist globally.

api.resetShopItemForPlayer(playerId, categoryKey, itemKey)

Remove this player's overrides, reverting them to the global item. Does nothing if no override exists.

api.configureShopCategoryForPlayer(playerId, categoryKey, config)

Set category properties (title, sort, redDot, etc.) for one player.

// VIP players get a discounted price
function applyVIPDiscount(playerId) {
  api.updateShopItemForPlayer(playerId, "myGame:weapons", "legendaryBow", {
    cost: 500,
    customTitle: "Legendary Bow (VIP Price!)",
    redDot: true
  });
}

// New player gets a one-time welcome item
api.createShopItemForPlayer(newPlayerId, "myGame:welcome", "starterPack", {
  image: "Chest",
  customTitle: "New Player Starter Pack",
  description: "A gift just for you!",
  cost: 0,
  canBuy: true
});

Callbacks

React to shop events in your game logic.

onPlayerBoughtShopItem

Fired after a player successfully completes a purchase. This is where all your game logic goes.

ParamTypeDescription
playerIdPlayerIdThe buying player
categoryKeyShopCategoryKeyCategory the item is in
itemKeyShopItemKeyThe item that was bought
itemBoughtShopItemResolved item (overrides applied, internals stripped)
userInputstring?The text/selection from userInput if configured
 
api.onPlayerBoughtShopItem = (playerId, categoryKey, itemKey, item, userInput) => {
  switch(itemKey) {
    case "healthPotion":
      healPlayer(playerId, 50);
      break;

    case "renameTribe":
      renameTribe(getTribeOf(playerId), userInput);
      break;

    case "sendGift":
      // userInput is the chosen playerId (player selector)
      giveItem(userInput, "Diamond", 1);
      break;
  }
};

openShop ()

Programmatically open the shop for a player from your script.

Signature

api.openShop(playerId, toggle, forceCategoryKey, onlyIfNonEmpty)
ParamTypeDescription
playerIdPlayerIdPlayer to open the shop for
toggleboolean?If true, closes the shop when already open
forceCategoryKeyPNull<ShopCategoryKey>Jump directly to a category tab
onlyIfNonEmptyboolean?Only open if the category (or shop) has items
 
// Open shop for a player
api.openShop(playerId, false);

// Open directly to the weapons category
api.openShop(playerId, false, "myGame:weapons");

// Toggle the shop open/closed
api.openShop(playerId, true);

// Only open if the category has items
api.openShop(playerId, false, "myGame:rewards", true);

Tribe Management Shop

api.configureShopCategory("tribe:manageTribe", {
  customTitle: "Manage Tribe",
  sortPriority: 50
});

api.createShopItem("tribe:manageTribe", "disbandTribe", {
  image: "tribeLeave.png",
  customTitle: [{ translationKey: "tribe:disbandTribe" }],
  description: [{ translationKey: "tribe:disbandTribeDescription" }],
  buyButtonText: [{ translationKey: "general:disband" }],
  sortPriority: 10,
  canBuy: true
});

api.createShopItem("tribe:manageTribe", "invitePlayer", {
  image: "tribeAddPlayer.png",
  customTitle: [{ translationKey: "tribe:invitePlayer" }],
  description: [{ translationKey: "tribe:invitePlayerToTribe" }],
  buyButtonText: [{ translationKey: "general:invite" }],
  sortPriority: 50,
  canBuy: true,
  userInput: { type: "player", excludedPlayers: [myPlayerId] }
});

api.onPlayerBoughtShopItem = (playerId, categoryKey, itemKey, item, userInput) => {
  if (itemKey === "disbandTribe") disbandTribe(playerId);
  if (itemKey === "invitePlayer") inviteToTribe(playerId, userInput);
};

};

Level-Gated Item with Per-Player Unlock

// Create globally (locked by default)
api.createShopItem("myGame:weapons", "dragonSword", {
  image: "Diamond Sword",
  imageColour: "#ff4400",
  customTitle: "🐉 Dragon Sword",
  description: "Requires level 50 to unlock.",
  cost: 2000,
  currency: "Gold Coin",
  canBuy: false,
  sortPriority: 5
});

// When a player reaches level 50, unlock it for them
function onPlayerLevelUp(playerId, newLevel) {
  if (newLevel >= 50) {
    api.updateShopItemForPlayer(playerId, "myGame:weapons", "dragonSword", {
      canBuy: true,
      description: "You've unlocked this legendary blade!",
      redDot: true
    });
  }
}
